The Linux Kernel

The Linux kernel is the main component in the Linux operating system. It is also the core communicator between a computerâ€™s hardware and its processes. It exists within the OS and controls the major functions of the hardware of all computers, phones, laptops and servers. The kernel has 4 jobs memory management which keeps track of the amount of memory used, what is being stored and where it is being stored. Process management which determines what processes are going to use the CPU, when they are going to use the CPU and how long the CPU will be used.
